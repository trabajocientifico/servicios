Contenido del Programa
Tres niveles · 12 horas cada uno
Desde la sintaxis básica de Python hasta modelos de machine learning y conexión con APIs de IA. Todo en Google Colaboratory, sin instalaciones.
1
Fundamentos de Python y Librerías
12 horas
  • Entorno Google Colab y Jupyter Notebooks
  • Variables, tipos de datos y estructuras
  • Condicionales, bucles y funciones
  • Manejo de archivos CSV, Excel y JSON
  • Introducción a NumPy: arrays y operaciones
  • Introducción a Pandas: Series y DataFrames
  • Lectura y escritura de datos desde la nube
  • Buenas prácticas de código con GitHub
2
Análisis de Datos con Pandas y NumPy
12 horas
  • Limpieza y transformación de datos sucios
  • Análisis exploratorio (EDA) completo
  • Agrupaciones, pivotes y operaciones avanzadas
  • Visualización con Matplotlib y Seaborn
  • Estadística descriptiva e inferencial con Python
  • Manejo de series de tiempo y datos geográficos
  • Integración con hojas de cálculo y bases de datos
  • Reporte automático en PDF y HTML
3
Machine Learning e IA Aplicada
12 horas
  • Introducción a Scikit-learn: flujo completo ML
  • Modelos de clasificación y regresión
  • Validación, métricas y ajuste de modelos
  • Árboles de decisión y Random Forest
  • Clustering: K-means y análisis de grupos
  • Conexión con APIs de IA (OpenAI, Anthropic)
  • Automatización de análisis con agentes de IA
  • Proyecto final: pipeline de datos completo con ML
